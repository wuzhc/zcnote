方法一: 使用 git remote add 命令

1# 本地仓库与远程仓库关联起来，再查看一下远程仓库情况

git remote add origin https://gitee.com/fox9916/gaoxTest.git

git remote -v

2# 使用如下命令再添加另一个远程仓库

git remote add sundyn http://192.168.100.54/git/gaoxing/gaoxTest.git
